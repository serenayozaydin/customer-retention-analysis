# Customer Retention Analysis

## Project Overview

This project analyzes customer retention behavior using SQL, Python and Pandas.

The goal is to identify customer retention patterns, analyze customer activity over time and generate business insights that can improve customer loyalty.

## Technologies Used

- Python
- Pandas
- SQL
- Jupyter Notebook

## Project Objectives

- Data cleaning and preprocessing
- Customer retention analysis
- Cohort analysis
- Customer segmentation
- Data visualization

## Status

Completed

## Dataset
Telco Customer Churn Dataset

## Key Findings

- Month-to-month customers had the highest churn rate (42.71%)
- Fiber optic customers churned more frequently (41.89%)
- Electronic check users showed the highest churn rate (45.29%)
- Customers without Online Security churned more often (41.77%)
- Customers without Tech Support churned more often (41.64%)
- Customers with higher monthly charges were more likely to churn

## Business Recommendations

- Encourage long-term contracts
- Promote Online Security and Tech Support packages
- Investigate customer dissatisfaction among Fiber Optic users
- Incentivize automatic payment methods
