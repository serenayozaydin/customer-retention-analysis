# Customer Retention Analysis

## Project Overview

This project analyzes customer retention behavior using SQL, Python and Pandas.

The goal is to identify customer retention patterns, analyze customer activity over time and generate business insights that can improve customer loyalty.

## Technologies Used

- Python
- Pandas
- SQL
- Jupyter Notebook

## Project Objectives

- Data cleaning and preprocessing
- Customer retention analysis
- Cohort analysis
- Customer segmentation
- Data visualization
